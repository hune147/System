# report
입력값은 16비트의 unsigned short 형태로 받아 처리<br/>
'printBinary'함수는 입력값을 2진수로 변환하여 출력하는 함수<br/>
'main'함수에서는 입력값을 16진수로 받아 'printBinary'함수를 호출하여 2진수로 출력<br/> 
출력 형식에 맞게 12번째 비트와 9번째 비트 뒤에는 콜론을 출력하도록 구현<br/>
